## 🎮 About Meta OS

Meta OS (TCN OS) is a futuristic, cyberpunk-themed Linux distribution inspired by **Kali Linux** and **Parrot Security OS**, designed for:

✨ **Cybersecurity Learning**  
🔨 **Ethical Hacking**  
🔍 **Digital Forensics**  
💻 **Programming & Development**  
🌐 **Network Analysis**  

### 🎨 Design Philosophy

Meta OS blends the aesthetics of:
- 🎬 Mr. Robot terminal style
- 🏙️ Cyberpunk 2077 UI
- 📺 Retro CRT monitors with scanlines & glitch effects
- 🟢 Matrix digital rain
- ⚡ Neon futuristic technology

---

## ⚡ Features

### 🖥️ Terminal Environment
- **Custom ZSH Shell** with cyberpunk themes
- **CRT Monitor Effects**: Scanlines, static noise, flickering, RGB separation
- **Animated Boot Sequence** with ASCII art
- **Matrix-style Particle Effects** in terminal background
- **Custom Prompt**: `[TUCCI@CYBER-NATION]:~#`
- **Neon Color Scheme**: Purple, Blue, Cyan, Red on black background

### 🛠️ Development Tools
- Python 3, Node.js, TypeScript
- Java, Rust, Go runtime environments
- VS Code with cyberpunk theme
- Git & GitHub integration
- Docker support

### 🔐 Security Tools
- **Wireshark** - Packet analysis
- **Nmap** - Network discovery
- **OpenVAS** - Vulnerability scanner
- **Metasploit Framework** - Penetration testing (authorized only)
- **Burp Suite Community** - Web security testing
- **John the Ripper** - Password cracking
- **Aircrack-ng** - Wireless security
- **Hashcat** - Hash cracking
- **Forensics Tools** - Log analyzers & data recovery

### 🎨 Customization
- Dark cyber wallpaper collection
- Custom system icons & sounds
- Animated login screen
- Cyber-themed system monitor dashboard
- CRT glitch animations & visual effects

## 🚀 Quick Start (Termux)

### Prerequisites
- Termux installed on Android
- Git installed
- Bash shell

### Installation

```bash
# Clone the repository
git clone https://github.com/TucciWrld/Meta-Spot.git
cd Meta-Spot

# Checkout development branch
git checkout tcn-os-development

# Run the main installation script
bash setup/install-meta-os.sh

# Initialize the cyberpunk environment
bash setup/init-cyberpunk.sh

# Start Meta OS
bash bin/start-meta-os.sh
```
